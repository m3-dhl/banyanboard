# Project Brief

## Project Overview

- **Name**: BanyanBoard
- **Description**: Self-hosted kanban board for small teams. Boards with columns, cards with titles/descriptions/due dates/labels, drag-and-drop between columns.
- **Goals**: Simple visual task tracking for 2–15 person teams without enterprise tool overhead
- **Scope**: MVP — boards, columns, cards, labels, multi-user, Docker Compose deployment
